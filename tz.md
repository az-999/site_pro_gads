Нам нужна платформа для автоматизации маркетинга, написанная с использованием Yii2.

1. Сгенерируйте несколько источников данных (CSV или JSON) в качестве входных данных:
- google_ads_keywords.csv — Site.pro ключевые слова, используемые в рекламе
- search_console_queries.csv — Site.pro ключевые слова, используемые в Search Console
- ahrefs_organic_keywords.csv — сайте.про ключевые слова, используемые в блогов Яндекса
- ahrefs_paid_keywords.csv — ключевые слова, используемые конкурентами (из Ahrefs)

2. Создайте систему, которая разрешает импорт этих файлов (должны приниматься CSV, JSON. Позже мы будем использовать API)
1) В системе должна быть административная панель, где отображаются все данные
2) Удалить неправильные ключевые слова
- Удалить мусор
- Удалить дубликаты и названия брендов
- Фильтровать по частоте

3. Подготовить ключевые слова для Google Ads:
- Удалить уже использованные и запрещенные ключевые слова
- Объединить дубликаты ключевых слов
- Сгруппировать ключевые слова по языкам
- Создать для них рекламу (на их языке и с правильным целевым URL) в формате Google Ads

4. Создать предварительный просмотр и файл для импорта в Google Ads

5. Предоставить URL, по которому мы можем протестировать рекламу: возможность загрузки, администрирования, предварительного просмотра и экспорта
